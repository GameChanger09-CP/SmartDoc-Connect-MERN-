source venv/bin/activate -> activate the virtual env ... 

sudo -u postgres psql -c "DROP DATABASE smartdoc_db;" -> delete old database 

sudo -u postgres psql -c "CREATE DATABASE smartdoc_db;" -> CREATE new database 

find . -path "*/migrations/*.py" -not -name "__init__.py" -delete
find . -path "*/migrations/*.pyc"  -delete
-> delete history files ... 

python manage.py makemigrations   -> Scans your models.py file and writes new instructions on how to build the database tables (Users, Documents, Departments).
python manage.py migrate -> Apply migrations ... 

python manage.py createsuperuser -> createsuperuser creates main admin physical entry ... 

python manage.py shell -c "from api.models import User; u = User.objects.get(username='admin'); u.role = 'Main_Admin'; u.kyc_status = 'Verified'; u.save(); print('Admin Role Fixed!')" -> fix main admin ... 

python manage.py runserver -> run the backend ...
